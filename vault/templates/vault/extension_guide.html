Extension Usage Guide

Learn how to use the CryptVault browser extension to securely autofill your credentials.

How to Use the CryptVault Extension

  1. Navigate to a Website

    Open your browser and go to the login page of the website where you want to use your credentials (e.g., https://example.com/login).

  2. Click the Extension Icon

    Find the CryptVault extension icon in your browser’s toolbar (usually top-right). Click it to open the extension’s interface.

  3. Enter CryptVault Credentials

    In the extension’s interface, enter your CryptVault application username and master password to authenticate and access your stored credentials.

  4. View Saved Credentials

    After authentication, the extension displays a list of usernames and passwords saved for the current website’s URL. Passwords are masked by default; click the “Show” button to reveal a password if needed.

  5. Autofill Credentials

    Select the desired username and password pair from the list. Click the “Autofill” button to automatically fill the login form on the website.
